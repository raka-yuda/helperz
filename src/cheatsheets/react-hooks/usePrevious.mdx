# usePrevious Hook

The `usePrevious` hook allows you to keep track of the previous value of a state or prop.

```jsx
import { useRef, useEffect } from 'react';

function usePrevious(value) {
  const ref = useRef();

  useEffect(() => {
    ref.current = value;
  }, [value]);

  return ref.current;
}
```
